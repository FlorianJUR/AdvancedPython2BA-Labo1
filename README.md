## Basic git
+ Open a folder on your computer
+ Open VScode with this folder and put <code>git clone <github repository link></code>
++ <code>git branch <branch name></code> (create a git branch, <code>git branch</code> to see all branches)
++ <code>git pull</code> (take the last push, by default main)
+ <code>git add <file name></code> or the + on VScode (<code>git status</code> to see all selected files 
+ <code>git commit -m <message></code> or commit button on VScode
++ <code>git checkout main</code> (we go on main)
++ <code>git merge <branch name></code> (update main with branche info)
+ <code>git push origin <branch name></code> (everyone can see and modify, it is now on Github)
+ <code>git pull</code> (take the last push, by default main)
